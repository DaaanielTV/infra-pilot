# Web Panel Implementation

Frontend + backend prototype for server panel functionality.

## Stack

- Vite + React (frontend)
- Convex (backend/data/auth)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Frontend source: `src/`
- Backend source: `convex/`
- Generated Convex client code should be regenerated locally via Convex tooling.
