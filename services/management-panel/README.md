# Web Panel Implementation (Cosmic Infra Branding)

Frontend + backend prototype for server panel functionality. Branded UI uses Cosmic Infra tokens and UI kit.

## Stack

- Vite + React (frontend)
- Convex (backend/data/auth)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Frontend source: `src/`
- Backend source: `convex/`
- Generated Convex client code should be regenerated locally via Convex tooling.
